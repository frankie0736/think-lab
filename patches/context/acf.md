## 输出要求

当涉及 {{topic}} 时，最终输出需要明确：

1. **字段设置**
   - 字段类型选择（文本、图片、关系、重复器等）
   - 字段配置建议（必填、默认值、条件显示）

2. **Taxonomy 设计**
   - 需要哪些分类法（层级 vs 标签型）
   - 每个 Taxonomy 的 meta 字段

3. **Post Types**
   - 需要哪些自定义文章类型
   - 各 Post Type 关联的字段组
   - Post Type 之间的关联关系

4. **数据关系**
   - 字段之间的依赖关系
   - 内容之间的引用方式
