# User Stories

## 主要故事

| 角色 | 目标 | 原因 |
|------|------|------|
| AI | 检测到模糊需求时发起提问 | 避免基于错误假设实现 |
| 前端 | 收到工具调用后渲染选择 UI | 让用户直观选择 |
| 用户 | 通过单选/多选回答问题 | 比打字更快更准确 |
| AI | 根据回答判断是否继续追问 | 直到信息完整 |

## 边界情况

| 情况 | 期望行为 |
|------|----------|
| 用户选"其他" | 允许自由输入 |
| 信息已足够 | AI 不再追问，直接执行 |
| 用户跳过 | AI 基于默认假设继续 |
