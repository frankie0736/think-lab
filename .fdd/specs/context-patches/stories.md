# User Stories

## 核心场景

**作为**使用思考助手的用户
**当**我讨论特定领域问题（如 WordPress ACF）
**我希望** AI 输出时自动包含该领域的关键要素提醒
**以便**不会遗漏重要的配置项

## 具体故事

1. **ACF 字段规划**
   - 用户问「我要做一个房产网站」
   - AI 检测到可能涉及 ACF → 注入 ACF context
   - 最终输出包含：字段类型建议、Taxonomy 设计、Post Type 规划

2. **非相关话题**
   - 用户问「帮我想想人生规划」
   - AI 检测不涉及任何 patch 领域 → 不注入
   - 正常输出思考助手内容
