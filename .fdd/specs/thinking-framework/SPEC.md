# Thinking Framework - 通用思考辅助工具

> 帮助任何人把一件事想清楚

## Intent

一个基于 6 阶段思考模型的通用采访工具，通过结构化提问引导用户从模糊想法走向清晰决策。

## 核心决策一览

| 维度 | 决策 |
|------|------|
| 目标用户 | 最终用户（任何想清楚一件事的人） |
| 阶段覆盖 | 完整 6 阶段：界定→发散→结构化→收敛→检验→反思 |
| 交互形式 | 统一使用 interview tool |
| 阶段感知 | 明确导航（用户知道在哪） |
| 阶段转换 | **AI 自行判断**（无回溯机制） |
| 问题数量 | 每轮 1-4 个问题 |
| 深度控制 | AI 自动判断 |
| 产出形式 | 单个 System Prompt |
| 最终输出 | Markdown 文档（问题定义 + 完整思维记录 + 结论 + 反思） |
