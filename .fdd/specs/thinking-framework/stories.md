# 用户故事

## 核心故事

**作为**任何想把一件事想清楚的人，
**我想要**一个结构化的思考辅助工具，
**以便**我能从模糊的想法走向清晰的决策。

## 场景示例

| 输入 | 期望产出 |
|------|---------|
| "我想转行做产品经理" | 问题定义 + 关键取舍 + 下一步行动 |
| "这两个技术方案怎么选" | 决策框架 + 利弊分析 + 推荐理由 |
| "我要不要搬去另一个城市" | 核心顾虑 + 价值排序 + 决策依据 |
| "今晚吃什么" | （简化流程）快速引导 + 推荐 |

## 非故事（不做）

- AI 不代替用户做决策，只帮用户"想清楚"
- AI 不提供专业建议（法律、医疗等），只提供思考结构
